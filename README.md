# Mini-poject-Salary-Analysis-Using-Numpy

## Project Description
This is a mini project created using Python and NumPy to analyze employee salary data.

## Tools & Technologies
- Python
- NumPy
- Jupyter Notebook (Anaconda)

## Analysis Done
- Total salary calculation
- Average salary
- Maximum and minimum salary
- Salaries greater than average

## Dataset
Employee salary data stored in a NumPy array.

Example:
[15000, 22000, 18000, 30000, 25000]

## How to Run
1. Open the project in PyCharm
2. Make sure NumPy is installed
3. Run the Python file or Jupyter Notebook

## Learning Outcome
- Learned NumPy arrays
- Understood numerical computations
- Performed basic data analysis
